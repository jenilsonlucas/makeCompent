NAVBAR

A navbar é usada para criar menus de navegação no topo da página com links dinâmicos definidos via atributos data-*.

1. Exemplo de Uso


        <div class="component-navbar"
        data-links="GitHub,Documentação"
        data-hrefs="https://github.com,https://seusite.com/docs"
        data-links-contents="GitHub,Docs"
        data-links-classes="link,btn-link">
        <h3 class="logo">makeComponent</h3>
        </div>

2. Atributos Explicados

  • data-links: Nomes das rotas (apenas para controle interno).
  • data-hrefs: Endereços dos links.
  • data-links-contents: Texto que aparece nos botões ou links.
  • data-links-classes: Classes extras para cada link.
