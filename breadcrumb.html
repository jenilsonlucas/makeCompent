BREADCRUMB

O componente breadcrumb é utilizado para indicar o caminho de navegação da página atual. Ele ajuda os usuários a entenderem em que parte do site estão e permite uma navegação rápida entre os níveis.

Você define no HTML da seguinte forma:


      <div 
       class="component-breadcrumb"
       data-contents="Início,Produtos,Detalhes"
      ></div>

2. Resultado

3. Atributos disponíveis

  • data-contents: Define os nomes de cada etapa no breadcrumb (separados por vírgula).
  • data-links: (opcional) Define os links correspondentes para cada etapa, também separados por vírgula.
