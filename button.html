BOTÕES

Botões na biblioteca são elementos interativos renderizados dinamicamente com base em classes como component-button ou component-gButton.

Você define no HTML algo como:

          
            <div 
            class='component-button'
            ></div>

E a biblioteca gera o botão estilizado conforme definido no componente JavaScript.

4. Mudança dos dados dinamicamente

Você pode modificar o texto ou comportamento dos botões usando atributos data-* ou com JavaScript.

          
            <div 
            class='component-button'
            data-component-button="Salvar"
            data-classes="botao-vermelho"
            ></div>
