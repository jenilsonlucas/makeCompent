Introdução

Esta biblioteca foi desenvolvida para criar e renderizar componentes HTML de forma dinâmica a partir de atributos personalizados (data-*) e classes padronizadas (component-*). O objetivo é simplificar a criação de elementos interativos e estruturados, como tabelas, slides, breadcrumbs, abas (tabs), formulários e outros, sem a necessidade de escrever manualmente todo o HTML de cada componente.

Através de um conjunto de funções reutilizáveis, a biblioteca /:

  • Identifica elementos no DOM que possuem classes iniciadas por component-.
  • Constrói dinamicamente o HTML correspondente, aplicando as classes e propriedades definidas.
  • Suporta atributos personalizados para personalização de conteúdo, aparência e comportamento.
  • Substitui automaticamente os elementos originais pelo componente renderizado.

Essa abordagem proporciona maior produtividade no desenvolvimento, padronização da interface e facilidade de manutenção do código, sendo ideal para projetos que demandam muitos elementos reutilizáveis com comportamentos semelhantes.

Instalação

Você pode instalar a biblioteca diretamente a partir do repositório no GitHub.

Abra o terminal no seu computador e digite o comando/:

          
            git clone https/://github.com/jenilsonlucas/makeCompent.git

copia todos os arquivos disponiveis no index.html e coloque no seu projecto!
