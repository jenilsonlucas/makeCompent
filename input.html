INPUTS

No sistema, inputs são componentes interativos renderizados dinamicamente com base em classes como component-input-box--text, component-input--radio, etc.

Você define no HTML assim:

<div class='component-input-box--text'></div>

E o JS transforma isso em um input estilizado completo.

Use data-* para alterar comportamentos e aparências:


 <div
  class='component-input-box--text'
  data-placeholder="Seu email"
  data-name="email"
  data-classes="input-email"
></div>
